# PyPlaysDinosaurGame
Step1: Run this code 
N.B: It is a OpenCV based code so you must have a webcam 
Step 2: the camera will open and start searching for a hand 
Step 3: To be specific it will be searching for a fist 
Step 4: after running the code also open the Dinosaur game in chrome 
Step 5 : place your fist in front of the camera 
Step 6: for close fist the dinosaur will move 
Step 7: for open fist / palm the dinosaur will jump
